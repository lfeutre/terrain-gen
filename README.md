## Installation

Just add it to your ``rebar.config`` deps:

```erlang
    {deps, [
        ...
        {terrain-gen, ".*", {git, "git@github.com:YOURNAME/terrain-gen.git", "master"}}
      ]}.
```

And then do the usual:

```bash
    $ rebar get-deps
    $ rebar compile
```
